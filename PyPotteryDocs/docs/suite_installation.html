Suite Installation

üöÄ PyPottery Suite Launcher

The easiest way to get started with PyPottery! Download the pre-packaged launcher with everything included - no Python installation required.

Quick Start

The PyPottery Suite Launcher is a standalone application that lets you download, install, and run all PyPottery tools from a single, user-friendly interface.

No Technical Knowledge Required

The launcher handles everything: Python environment, dependencies, and app management. Just download, extract, and run!

Download

Choose the package for your operating system:

Platform Download Size
Windows 10/11 (64-bit) PyPottery-Launcher-Windows.zip ~52 MB
macOS Intel (x86_64) PyPottery-Launcher-macos-x86_64.zip ~22 MB
macOS Apple Silicon (M1/M2/M3) PyPottery-Launcher-macos-arm64.zip ~22 MB
Linux (x86_64) PyPottery-Launcher-linux-x86_64.zip ~128 MB

Installation

Windows

  1. Download the Windows package from the link above
  2. Extract the ZIP file to a folder of your choice (e.g., C:\PyPottery)
  3. Run PyPottery.bat by double-clicking it
üí° Tip

You can create a desktop shortcut to PyPottery.bat for quick access!

macOS

  1. Download the appropriate package for your Mac:

    • Intel Mac: macos-x86_64 version
    • Apple Silicon (M1/M2/M3): macos-arm64 version

  2. Extract the ZIP file

  3. Open Terminal and navigate to the extracted folder

  4. Run the installer (first time only):

    ./install.sh

  5. Launch PyPottery:

    ./PyPottery.sh
First Run on macOS

The first time you run the launcher, macOS may ask for permission. Right-click on the script and select ‚ÄúOpen‚Äù to bypass Gatekeeper.

Linux

  1. Download the Linux package

  2. Extract the ZIP file:

    unzip PyPottery-Launcher-linux-x86_64-v1.0.0.zip
cd PyPottery-Launcher-linux-x86_64

  3. Run the installer (first time only):

    chmod +x install.sh PyPottery.sh
./install.sh

  4. Launch PyPottery:

    ./PyPottery.sh
